# 🎮 Personal Quiz Game

A responsive web-based quiz application inspired by Kahoot, built using **HTML, CSS, and JavaScript**. The project includes a login system, timed questions, score tracking, animations, and local high-score storage.

## 🚀 Features

- 🔐 Login authentication page
- ⏱️ 15-second timer for each question
- ✅ Instant answer feedback
- 🎉 Confetti animation for correct answers
- 📊 Live score tracking
- 📈 Progress bar
- ⏭️ Skip question functionality
- 🏆 Local high-score leaderboard using LocalStorage
- 🔄 Restart and Play Again options
- 📱 Fully responsive design
- 🚪 Logout functionality

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## ▶️ How to Run

1. Download or clone the repository:

```bash
git clone https://github.com/your-username/personal-quiz-game.git
```

2. Open the project folder.

3. Launch `login.html` in your browser.

4. Login using the demo credentials and start playing.
